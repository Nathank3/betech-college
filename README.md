# Betech Computer College

A modern, tech-forward landing page for Betech Computer College, located in
Wote, Makueni. Built with a focus on high conversion, aesthetics, and mobile
responsiveness.

## Features

- **Modern Dark Theme:** Sleek glassmorphism and neon accents.
- **Course Catalog:** Categorized courses with "View All" functionality.
- **Interactive Chatbot:** Instant guidance for prospective students.
- **Google Maps Integration:** Easy location lookup for the Wote Campus.
- **Mobile Responsive:** Optimized for all devices.

## Tech Stack

- **Backend:** Node.js with Express (Serving EJS templates)
- **Frontend:** HTML5, Vanilla CSS, Tailwind CSS
- **Templating:** EJS (Embedded JavaScript)

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```
